## 技术文档

### ESP32-S3 开发板

- [ESP32 开发板资料页](https://www.xinlucity.com/?s=resourcedetail/index/id/61.html)
  开发板在线资料入口，包含原理图、参考例程、软件下载、引脚定义、出厂固件和注意事项等。

- [ESP32-S3原理图.pdf](技术文档/ESP32-S3原理图.pdf)
  开发板的完整电路原理图，能看到：
  - 每个引脚连接了什么

  - 电源电路设计
  - 复位和下载电路
  
  **什么时候需要看**：引脚冲突、外设不工作、想自己画板子

### MG4010-i10 电机

- [MG4010-i10电机手册.pdf](技术文档/MG4010-i10电机手册.pdf)
  电机的详细规格书：
  - 技术参数（电压、电流、转速、扭矩）
  - 机械尺寸
  - 安装方式
  - 使用注意事项

- [MG4010-i10-R485协议.pdf](技术文档/MG4010-i10-R485协议.pdf)
  电机的通信协议文档：
  - RS485 接线方法
  - 厂家私有帧格式，帧头为 `0x3E`
  - 命令格式（运行、停止、速度控制、位置控制）
  - 反馈数据格式
  - 注意：该协议不是 Modbus-RTU 寄存器协议

### RS485/供电转接板

- [转接板原理图.pdf](../转接板原理图.pdf)
  课程配套的 RS485 通信与电机供电转接板原理图。教程中的 MG4010-i10 电机实验建议优先使用这块板，减少散装 RS485 模块、独立电源和电机端子之间的接线错误。

- [电机转接板接线示意图.svg](接线图/电机转接板接线示意图.svg)
  面向上机接线的示意图，标出了 ESP32 供电、TTL-485 调试口、电机 6pin 接口、24V 输入和 MG90S 舵机接口。

- [电机转接板实物图.png](接线图/电机转接板实物图.png)
  转接板正面图片，用于核对接口位置、丝印和防呆方向。

  **什么时候需要看**：上机接线、电机无回复、确认底部 6pin 的 `B/A/GND/GND/24V/24V`、核对 TTL-485 调试口 `T/R` 与 ESP32 GPIO17/GPIO18 的连接关系。

## 驱动程序

### [CH341SER.EXE](驱动程序/CH341SER.EXE)

WCH CH340/CH341 USB 转串口芯片的 Windows 驱动。

**什么时候需要装**：
- 插上板子，设备管理器里没有 COM 口
- 设备管理器里出现黄色感叹号的未知设备
- 提示"无法识别的 USB 设备"

**怎么装**：
1. 双击 `CH341SER.EXE`
2. 点"安装"
3. 拔掉板子重新插上
4. 设备管理器里应该能看到 `USB-SERIAL CH340 (COMx)`

**注意**：
- Windows 10/11 一般能自动识别，不需要手动装
- 如果板子用的是 CH343 芯片，需要装 CH343 的驱动（不在这个包里，去 [WCH 官网](https://www.wch.cn/downloads/CH343SER_EXE.html) 下）
- macOS 用户看 `教程文档/02_install_macos.md`
