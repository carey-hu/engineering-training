# 10 教程审阅报告

> 本报告面向教师和维护者，记录本仓库教程与例程的规范性、正确性和改进建议。学生第一次学习应先看 [`00_learning_path.md`](00_learning_path.md)。

## 1. 总体结论

当前教程可以作为 ESP32-S3、普通 PWM 舵机、MG4010-i10 RS485 电机的入门训练材料使用。内容覆盖了环境搭建、板卡参数、常见问题、硬件接线、舵机控制、电机协议、网页控制和最终整合，主题完整。经过本轮调整后，学生入口更清晰：先用板载普通 LED 做无接线 GPIO 输出验证，再学习串口、舵机、RS485 电机，最后整合两电机四舵机。

主要优点：

- 使用了国内 ESP32 板卡包镜像，适合国内课堂环境。
- 第一个实验改为 GPIO2 板载普通 LED，无需外接元件，同时直接教学 `pinMode()` 和 `digitalWrite()`。
- 舵机例程使用 ESP32-S3 Arduino-ESP32 3.x 的 `ledcAttach(pin, freq, resolution)` 写法。
- 电机例程明确说明 MG4010-i10 RS485 协议不是 Modbus-RTU，避免学生按寄存器协议排错。
- 已把课程配套 RS485/供电转接板纳入学习路线、硬件资料、电机指南和电机例程说明。
- RS485 帧格式、头校验、数据校验、小端序和减速比换算在代码中保持一致。
- 最终整合例程使用电机 ID 参数和四舵机数组，结构上适合作为项目模板。

主要风险：

- README 信息量较大，学生容易从中间章节开始跳读，漏掉“先板载普通 LED、再串口、再舵机、再电机”的顺序。
- 电机第一次上电有机械风险，必须反复强调低速、小角度、空载、可立即断电。
- 多电源共地是最容易出错的地方，应在课堂检查中作为必查项。
- 如果没有安装 Arduino CLI，本仓库无法在命令行中自动完成编译验证；需要在 Arduino IDE 中逐个打开示例编译。

## 4. MG4010-i10 协议核对

代码中的协议处理符合教程内协议描述：

```text
帧头: 0x3E
命令字: CMD
电机 ID: 1~32
数据长度: N
头校验: 帧头 + 命令字 + ID + 数据长度，取低 8 位
数据区: 小端序
数据校验: 数据区所有字节相加，取低 8 位
```

已覆盖的命令：

| 命令 | 用途 | 代码位置 |
|------|------|----------|
| `0x80` | 电机关闭 | 06 |
| `0x81` | 电机停止 | 06、07、08、09、10 |
| `0x88` | 电机运行 | 06、07、08、09、10 |
| `0x92` | 读取多圈角度 | 06、08、09 |
| `0x9B` | 清除错误 | 06 |
| `0x9C` | 读取状态 2 | 06、07、09 |
| `0xA2` | 速度闭环 | 07、09、10 |
| `0xA4` | 多圈位置闭环 2 | 08、09 |
| `0x19` | 设置零点到 ROM | 06 |

教学注意点：

- `0x19` 会写入 ROM/Flash，不应作为普通运行命令反复调用。
- 速度和位置示例中，学生输入的是输出轴数值；代码会乘以 i10 减速比后发送给电机。
- 两个电机共用一条 RS485 总线时，ID 必须不同，否则读状态时容易总线冲突。

## 5. 建议的课堂检查点

每组学生上机时建议教师按以下顺序检查：

1. Arduino IDE 板卡和端口是否正确。
2. 板载普通 LED 闪烁是否成功。
3. 串口监视器是否有 `115200` 输出。
4. 舵机是否使用独立 5V，是否共地。
5. 电机是否使用独立 12V~24V，是否通过转接板或 RS485 模块可靠共地。
6. 使用转接板时是否按丝印确认输入端子上正下负、TTL-485 调试口 `T/R`、电机 6pin 的 `B/A/GND/GND/24V/24V`，并确认 ESP32 不需要接 GPIO16。
7. 两个电机 ID 是否分别为 1 和 2。
8. 第一次电机速度是否不超过输出轴 `30 dps` 或 `60 dps`。
9. 四舵机同时动作时电源是否掉压。
10. 最终整合前是否分别跑通过单舵机和单电机例程。

## 6. 后续可继续增强

建议后续增加但不是当前必须项：

- 给每个示例增加一张接线图图片，降低读文字接线的错误率。
- 安装 `arduino-cli` 后增加批量编译脚本，课堂发版前自动验证所有 `.ino`。
- 给 MG4010 增加“设置/读取电机 ID”的独立安全教程，但必须要求单电机接线，避免多个同 ID 设备同时响应。
- 给最终机器人项目增加急停输入，例如实体按键或网页急停按钮。
- 为舵机角度增加每个关节的软限位配置，避免装机后机械卡死。
