# 05 教师课前检查清单

## 硬件

- [ ] ESP32-S3 开发板型号统一
- [ ] 每块板能正常识别 COM 口
- [ ] 每组至少一根确认可用的数据线
- [ ] 外接 LED、220Ω 电阻、杜邦线
- [ ] 若控制舵机/马达，准备独立 5V 电源并共地

## 软件

- [ ] Arduino IDE 安装包已下载
- [ ] CH340/CH343 驱动已下载
- [ ] ESP32-S3 板卡包安装成功
- [ ] 示例代码能编译上传
- [ ] 教师机可演示 Wi-Fi AP 网页控制

## 学生常见问题预案

- [ ] 无 COM 口：换线、装驱动、看设备管理器
- [ ] 上传失败：BOOT 键、降速到 115200
- [ ] 串口无输出：USB CDC On Boot 开启
- [ ] 编译失败：检查中文路径、板卡包版本
- [ ] 网页控制卡顿：检查 delay、滑块请求频率
