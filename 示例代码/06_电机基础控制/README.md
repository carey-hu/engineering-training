# MG4010 电机基础控制（Modbus-RTU）

## 功能说明

通过 RS485 串口 + Modbus-RTU 协议控制 MG4010 电机，实现使能、失能、零点设置和状态读取。

## 硬件需求

1. ESP32-S3 开发板
2. MG4010-i10 电机及驱动器（RS485 版本）
3. RS485 转换模块（MAX485 或 MAX3485）
4. 24V 直流电源（电机供电）
5. 杜邦线

## 接线

### ESP32-S3 → RS485 模块

| ESP32-S3 | RS485 模块 | 说明 |
|----------|-----------|------|
| GPIO17 (TX2) | DI | 发送数据 |
| GPIO18 (RX2) | RO | 接收数据 |
| GPIO16 | DE + RE | 收发控制（两个引脚连一起） |
| 3.3V | VCC | 电源 |
| GND | GND | 地线 |

### RS485 模块 → MG4010 驱动器

| RS485 模块 | MG4010 驱动器 |
|-----------|-------------|
| A+ | A+ (485+) |
| B- | B- (485-) |

### 电源

- ESP32-S3：USB 供电
- RS485 模块：3.3V 或 5V（看模块规格）
- MG4010 驱动：24V DC 外部电源

**必须共地：ESP32 GND — RS485 GND — 电机驱动 GND 连在一起。**

## 协议

Modbus-RTU，9600 波特率，8N1，从站地址 1。

| 寄存器 | 功能 | 说明 |
|--------|------|------|
| 0x0001 | 控制字 | 0x0001=使能, 0x0000=停机, 0x0002=设零点 |

## 串口命令

打开串口监视器（115200），输入：

| 命令 | 功能 |
|------|------|
| `e` | 电机使能 |
| `d` | 电机失能 |
| `z` | 设置零点 |
| `s` | 读取状态 |
| `h` | 帮助 |

## 使用步骤

1. 按接线图连接所有硬件，确保共地
2. 接 24V 电源到电机驱动器
3. 打开 Arduino IDE，选 ESP32S3 Dev Module
4. USB CDC On Boot: Enabled
5. 上传程序，打开串口监视器（115200）
6. 输入 `e` 使能，`s` 读状态，`d` 失能
