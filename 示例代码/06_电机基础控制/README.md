# MG4010-i10 RS485 基础控制

## 功能说明

通过 ESP32-S3、课程配套 RS485/供电转接板或通用 RS485 模块，以及厂家私有 RS485 协议控制 MG4010-i10 电机，实现运行、停止、关闭、清除错误、读取状态和读取多圈角度。

注意：本电机的厂家协议不是 Modbus-RTU。示例代码使用帧头 `0x3E`、命令字和校验和。

## 硬件需求

1. ESP32-S3 开发板
2. MG4010-i10 电机（RS485 版本）
3. RS485/供电转接板（推荐）或 3.3V 兼容 RS485 转换模块
4. 12V~24V 直流电源，首次测试建议使用 24V 2A 以上
5. 杜邦线

## 接线

推荐优先使用课程配套 RS485/供电转接板。按板子丝印连接：

| ESP32-S3 | 转接板 | 说明 |
|----------|--------|------|
| 5V | USB调试 5V | 转接板逻辑供电 |
| GND | USB调试 G | 共地 |
| GPIO17 (TX) | TTL-485 调试 T | 发送数据，TX 接 T |
| GPIO18 (RX) | TTL-485 调试 R | 接收数据，RX 接 R |

| 电机/电源 | 转接板 |
|----------|--------|
| 24V 电源正极 | 输入端子上孔 |
| 24V 电源负极 | 输入端子下孔 |
| MG4010 B/L | 电机 6pin 第 1 pin：B |
| MG4010 A/H | 电机 6pin 第 2 pin：A |
| MG4010 V- | 电机 6pin 第 3、4 pin：GND |
| MG4010 V+ | 电机 6pin 第 5、6 pin：24V |

如果不用转接板，再按下面的通用 RS485 模块接线：

| ESP32-S3 | RS485 模块 | 说明 |
|----------|------------|------|
| GPIO17 | DI | 发送数据 |
| GPIO18 | RO | 接收数据 |
| GPIO16 | DE + RE | 收发方向控制 |
| 3.3V | VCC | 模块供电 |
| GND | GND | 共地 |

课程配套转接板已在板上处理 RS485 收发方向，ESP32 不需要接 GPIO16。只有使用通用 RS485 模块时，才需要 GPIO16 控制 DE/RE。接线前仍要看清开发板丝印，不要把信号线误接到 3.3V、5V 或 GND。

| RS485 模块 | MG4010-i10 |
|------------|------------|
| A | A/H |
| B | B/L |

电机电源：

- V+ 接电源正极
- V- 接电源负极
- ESP32 GND、RS485/供电转接板或 RS485 模块 GND、电机 V- 必须共地

## 协议参数

```text
波特率：115200
数据格式：8N1
默认 ID：1
帧头：0x3E
```

## 串口命令

打开串口监视器，波特率选择 115200。

| 命令 | 功能 |
|------|------|
| `r` | 电机运行 |
| `t` | 电机停止 |
| `o` | 电机关闭 |
| `c` | 清除错误 |
| `s` | 读取状态 2 |
| `a` | 读取多圈角度 |
| `z` | 设置当前位置为零点，写入 ROM，谨慎使用 |
| `h` | 显示帮助 |

## 使用步骤

1. 断电状态下完成接线。
2. 确认电源正负极、A/B、共地无误。
3. 上传程序，打开串口监视器。
4. 给电机上电。
5. 先输入 `s` 读取状态，再输入 `r` 运行。
6. 需要停止时输入 `t`。
