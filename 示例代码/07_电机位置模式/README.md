# MG4010-i10 多圈位置闭环控制

## 功能说明

使用厂家 RS485 命令 `0xA4` 控制多圈位置，并同时设置速度限制。本例程实现了软件零点功能，上电后自动将当前位置设为 0 点。

**关键换算（MG4010-i10 实测修正版）：**
```text
角度：按输出轴角度处理，不乘减速比
协议位置原始值 = 输出轴角度 × 1 × 100

速度：按电机侧速度处理，需乘减速比
协议速度原始值 = 输出轴最大速度 dps × 10 × 100
```

## 硬件

接线和供电同 `05_电机基础控制`。推荐优先使用课程配套 RS485/供电转接板：GPIO17(TX) 接转接板 TTL-485 调试口 T，GPIO18(RX) 接 R，ESP32 5V/GND 接 USB调试 5V/G。转接板不需要 GPIO16。

如果使用通用 RS485 模块，再按下表接线。

| ESP32-S3 | RS485 模块 | 说明 |
|----------|------------|------|
| GPIO17 | DI | 发送数据 |
| GPIO18 | RO | 接收数据 |
| GPIO16 | DE + RE | 收发方向控制 |
| GND | GND | 与电机电源负极共地 |

GPIO17、GPIO18、GPIO16 已根据 ESP32-S3 开发板原理图确认从排针引出。课程转接板只用 GPIO17/GPIO18；通用模块才额外使用 GPIO16。

## 协议

```text
帧头：0x3E
命令：0xA4
数据长度：12
数据 0~7：int64_t angleControl，小端序，单位 0.01°
数据 8~11：uint32_t maxSpeed，小端序，单位 0.01dps
波特率：115200
```

## 串口命令

| 命令 | 功能 |
|------|------|
| `r` | 读取当前角度 |
| `z` | 将当前位置设为软件零点 |
| `v 30` | 设置输出轴最大速度为 30 dps |
| `30` | 输出轴转到软件坐标 30° |
| `+30` | 在当前目标基础上增加 30° |
| `-30` | 在当前目标基础上减少 30° |
| `0` | 回到软件零点 |
| `t` | 停止 |
| `off` | 关闭电机 |

## 使用步骤

1. 接线、供电同 05。
2. 上传程序，打开串口监视器，程序自动将当前位置设为软件零点。
3. 输入 `v 30` 设置较低移动速度。
4. 输入 `30`、`60`、`90` 逐步测试位置控制。
5. 输入 `0` 回到零点。
6. 输入 `t` 停止。

## 软件零点说明

- 上电后自动将当前位置定义为软件 0 点
- 串口输入 `z` 可重新设置当前位置为零点
- 软件零点不写入 ROM，断电后需重新设置
- 适合快速测试和调试，无需每次都找机械零位
