# MG4010-i10 速度闭环控制

## 功能说明

使用厂家 RS485 命令 `0xA2` 控制电机速度。串口中输入的是输出轴速度，单位为 dps（度/秒）。

MG4010-i10 减速比为 1:10，代码会自动换算为协议中的电机侧速度：

```text
协议原始值 = 输出轴速度 dps × 10 × 100
```

## 硬件

接线和供电同 `06_电机基础控制`。推荐优先使用课程配套 RS485/供电转接板：GPIO17(TX) 接转接板 TTL-485 调试口 T，GPIO18(RX) 接 R，ESP32 5V/GND 接 USB调试 5V/G。转接板不需要 GPIO16。

如果使用通用 RS485 模块，再按下表接线。

| ESP32-S3 | RS485 模块 | 说明 |
|----------|------------|------|
| GPIO17 | DI | 发送数据 |
| GPIO18 | RO | 接收数据 |
| GPIO16 | DE + RE | 收发方向控制 |
| GND | GND | 与电机电源负极共地 |

GPIO17、GPIO18、GPIO16 已根据 ESP32-S3 开发板原理图确认从排针引出。课程转接板只用 GPIO17/GPIO18；通用模块才额外使用 GPIO16。

## 协议

```text
帧头：0x3E
命令：0xA2
数据：int32_t speedControl，小端序，单位 0.01dps
波特率：115200
```

## 串口命令

| 命令 | 功能 |
|------|------|
| `36` | 设置输出轴速度为 36dps |
| `-36` | 反方向 36dps |
| `+` / `-` | 加减 10dps |
| `t` | 停止 |
| `s` | 读取状态 |
| `h` | 帮助 |

建议首次测试从 `30` 或 `60` 开始，不要直接输入很大的速度。

## 使用步骤

1. 接线、供电同 06。
2. 上传程序，打开串口监视器。
3. 输入 `30` 回车，观察输出轴缓慢转动。
4. 输入 `-30` 测试反方向。
5. 输入 `t` 停止。
