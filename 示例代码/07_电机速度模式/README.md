# MG4010 闭环速度模式（Modbus-RTU）

## 功能说明

闭环速度控制：设定目标转速（RPM），电机自动维持恒定速度。适合传送带、风扇、搅拌器等恒速应用。

## 硬件

同 06_电机基础控制。

## 协议

Modbus-RTU，9600，8N1，从站地址 1。

| 寄存器 | 功能 |
|--------|------|
| 0x0001 | 控制字（0x0001=使能） |
| 0x0002 | 目标速度（RPM） |

## 串口命令（115200）

| 命令 | 功能 |
|------|------|
| `500` | 设置目标速度 500 RPM |
| `+` / `-` | 加减 10 RPM |
| `t` | 停止 |
| `s` | 读取状态 |
| `h` | 帮助 |

速度范围：0 ~ 3000 RPM。

## 使用步骤

1. 接线、供电同 06
2. 上传程序，打开串口监视器
3. 输入目标速度值（如 `500`）回车
4. 电机启动并以 500 RPM 恒速运行
5. 输入 `+` / `-` 微调，`t` 停止
