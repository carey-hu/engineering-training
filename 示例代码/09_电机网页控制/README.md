# MG4010 电机网页综合控制（Modbus-RTU）

## 功能说明

WiFi AP 模式 + 网页控制，支持速度模式和位置模式切换，实时显示电机状态。

## 硬件

同 06_电机基础控制。

## 使用步骤

1. 接线、供电同 06
2. 上传程序
3. 手机/电脑连 WiFi：`ESP32S3_MOTOR`，密码 `12345678`
4. 浏览器打开 `http://192.168.4.1`
5. 选择模式，滑块或输入值，点击按钮执行

## 控制模式

- **速度模式**：滑块设 RPM（0 ~ 3000），电机恒速运行
- **位置模式**：滑块设角度（-4500° ~ 4500°），可设最大移动速度
- **紧急停止**：立即停机

## 注意事项

- 电机驱动需要 24V 外部电源
- 必须共地
- A+/B- 不能接反
