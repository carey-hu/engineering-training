# MG4010-i10 电机网页综合控制

## 功能说明

ESP32-S3 建立 Wi-Fi 热点，手机或电脑连接后通过网页控制 MG4010-i10 电机。示例支持：

- 速度闭环控制（厂家命令 `0xA2`）
- 多圈位置闭环控制（厂家命令 `0xA4`）
- 停止电机（厂家命令 `0x81`）
- 状态和角度刷新（厂家命令 `0x9C`、`0x92`）

网页上输入的速度和角度均指输出轴，代码内部按 i10 减速比换算。

## 硬件

接线和供电同 `06_电机基础控制`。推荐优先使用课程配套 RS485/供电转接板；如果使用通用 RS485 模块，再按下表接线。

| ESP32-S3 | RS485 模块 | 说明 |
|----------|------------|------|
| GPIO17 | DI | 发送数据 |
| GPIO18 | RO | 接收数据 |
| GPIO16 | DE + RE | 收发方向控制 |
| GND | GND | 与电机电源负极共地 |

GPIO17、GPIO18、GPIO16 已根据 ESP32-S3 开发板原理图确认从排针引出，可以按表格直接插线。

## 使用步骤

1. 接线、供电同 06。
2. 上传程序。
3. 手机或电脑连接 Wi-Fi：`ESP32S3_MOTOR`，密码 `12345678`。
4. 浏览器打开 `http://192.168.4.1`。
5. 先用较小速度或角度测试，例如速度 `30dps`、位置 `30°`。

## 控制模式

- 速度模式：设置输出轴速度，范围 `-360dps` 到 `360dps`。
- 位置模式：设置输出轴目标角度，范围 `-720°` 到 `720°`，并设置最大速度。
- 停止电机：发送停止命令，停止当前动作。

## 注意事项

- 电机必须使用独立电源。
- ESP32、RS485/供电转接板或 RS485 模块和电机电源负极必须共地。
- A/B 接反会导致没有回复。
- 网页控制前建议先运行 06、07、08，确认串口控制正常。
