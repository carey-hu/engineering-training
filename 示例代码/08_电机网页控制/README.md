# MG4010-i10 电机网页综合控制

## 功能说明

ESP32-S3 建立 Wi-Fi 热点，手机或电脑连接后通过网页控制 MG4010-i10 电机。示例支持：

- 速度闭环控制（厂家命令 `0xA2`）
- 多圈位置闭环控制（厂家命令 `0xA4`）
- 停止电机（厂家命令 `0x81`）
- 状态和角度刷新（厂家命令 `0x9C`、`0x92`）

网页上输入的速度单位为 RPM（转/分钟）和角度均指输出轴，代码内部按 i10 减速比换算。

## 硬件

接线和供电同 `05_电机基础控制`。推荐优先使用课程配套 RS485/供电转接板：GPIO17(TX) 接转接板 TTL-485 调试口 T，GPIO18(RX) 接 R，ESP32 5V/GND 接 USB调试 5V/G。转接板不需要 GPIO16。

如果使用通用 RS485 模块，再按下表接线。

| ESP32-S3 | RS485 模块 | 说明 |
|----------|------------|------|
| GPIO17 | DI | 发送数据 |
| GPIO18 | RO | 接收数据 |
| GPIO16 | DE + RE | 收发方向控制 |
| GND | GND | 与电机电源负极共地 |

GPIO17、GPIO18、GPIO16 已根据 ESP32-S3 开发板原理图确认从排针引出。课程转接板只用 GPIO17/GPIO18；通用模块才额外使用 GPIO16。

## 使用步骤

1. 接线、供电同 05。
2. 上传程序。
3. 手机或电脑连接 Wi-Fi：`ESP32S3_MOTOR`，密码 `12345678`。
4. 浏览器打开 `http://192.168.4.1`。
5. 先用较小速度或角度测试，例如速度 `5 RPM`、位置 `30°`。

## 控制模式

### 速度模式
- 拖动滑块或输入数字设置输出轴转速
- 范围：`-60 RPM` 到 `60 RPM`
- 正值顺时针，负值逆时针

### 位置模式
- **拖动滑块即时控制**：滑动到哪转到哪（-360° 到 +360°）
- 调整运行速度：1-120 RPM
- 快捷按钮：-180°、归零、+180°
- 采用软件零点，上电时当前位置为 0°

### 停止电机
- 点击"停止电机"按钮，立即停止当前动作

## 特色功能

- **位置滑块实时控制**：拖动滑块，电机立即跟随到目标位置
- **150ms 防抖**：避免拖动时频繁发送命令
- **软件零点**：上电自动记录当前位置为 0°，无需机械复位
- **实时状态显示**：每500ms 刷新转速、角度、温度

## 注意事项

- 电机必须使用独立 24V 电源
- ESP32、RS485/供电转接板或 RS485 模块和电机电源负极必须共地
- A/B 接反会导致没有回复
- 网页控制前建议先运行 05、06、07，确认串口控制正常
- 首次使用建议从低速（5-10 RPM）开始测试
