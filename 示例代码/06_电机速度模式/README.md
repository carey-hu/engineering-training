# MG4010-i10 速度闭环控制

## 功能说明

使用厂家 RS485 命令 `0xA2` 控制电机速度。串口中输入的是输出轴转速，单位为 RPM（转/分钟），更符合日常习惯。

代码会自动将 RPM 转换为 dps（度/秒），再换算为协议中的电机侧速度：

```text
RPM → dps：1 RPM = 6 dps（360度/60秒=6）
协议原始值 = 输出轴 dps × 10（减速比） × 100
```

MG4010-i10 减速比为 1:10。

## 硬件

接线和供电同 `05_电机基础控制`。推荐优先使用课程配套 RS485/供电转接板：GPIO17(TX) 接转接板 TTL-485 调试口 T，GPIO18(RX) 接 R，ESP32 5V/GND 接 USB调试 5V/G。转接板不需要 GPIO16。

如果使用通用 RS485 模块，再按下表接线。

| ESP32-S3 | RS485 模块 | 说明 |
|----------|------------|------|
| GPIO17 | DI | 发送数据 |
| GPIO18 | RO | 接收数据 |
| GPIO16 | DE + RE | 收发方向控制 |
| GND | GND | 与电机电源负极共地 |

GPIO17、GPIO18、GPIO16 已根据 ESP32-S3 开发板原理图确认从排针引出。课程转接板只用 GPIO17/GPIO18；通用模块才额外使用 GPIO16。

## 协议

```text
帧头：0x3E
命令：0xA2
数据：int32_t speedControl，小端序，单位 0.01dps
波特率：115200
```

## 串口命令

| 命令 | 功能 |
|------|------|
| `30` | 设置输出轴转速为 30 RPM（每分钟30圈）|
| `-30` | 反方向 30 RPM |
| `+` / `-` | 加减 2 RPM |
| `t` | 停止 |
| `s` | 读取状态 |
| `e` | 清除错误 |
| `h` | 帮助 |

建议首次测试从 `5` 或 `10` RPM 开始，不要直接输入很大的速度。

## 使用步骤

1. 接线、供电同 05。
2. 上传程序，打开串口监视器。
3. 输入 `5` 回车，观察输出轴缓慢转动（每分钟5圈）。
4. 输入 `-5` 测试反方向。
5. 输入 `t` 停止。

## RPM vs dps 对照

方便理解：
- 5 RPM = 30 dps（慢速）
- 10 RPM = 60 dps（中速）
- 30 RPM = 180 dps（快速）
- 60 RPM = 360 dps（最大速度）
