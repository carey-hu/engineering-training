# MG4010 闭环位置模式（Modbus-RTU）

## 功能说明

闭环位置控制：设定目标角度，电机自动转到指定位置。适合机械臂关节、云台、分度盘等定位应用。

## 硬件

同 06_电机基础控制。

## 协议

Modbus-RTU，9600，8N1，从站地址 1。

| 寄存器 | 功能 |
|--------|------|
| 0x0001 | 控制字（0x0003=位置模式） |
| 0x0003 | 目标位置高 16 位 |
| 0x0004 | 目标位置低 16 位（32 位位置值，0.01° 单位） |
| 0x0005 | 位置模式速度（RPM） |

位置单位 0.01°（36000 = 一圈）。

## 串口命令（115200）

| 命令 | 功能 |
|------|------|
| `180` | 转到 180° |
| `p 180` | 同上 |
| `s 200` | 设置位置模式速度 200 RPM |
| `0` | 归零 |
| `360` | 正向转一圈 |
| `-360` | 反向转一圈 |
| `t` | 停止 |
| `r` | 读取位置 |
| `h` | 帮助 |

角度范围：-4500° ~ +4500°（±12.5 圈）。

## 使用步骤

1. 接线、供电同 06
2. 上传程序，打开串口监视器
3. `s 200` 先设置移动速度
4. 输入目标角度（如 `180`）回车
5. 电机转到 180° 位置
6. 用 `0`、`360`、`-360` 快捷测试
