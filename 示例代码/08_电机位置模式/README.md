# MG4010-i10 多圈位置闭环控制

## 功能说明

使用厂家 RS485 命令 `0xA4` 控制多圈位置，并同时设置速度限制。串口中输入的是输出轴角度，代码会根据 i10 减速比自动换算为电机侧角度。

```text
协议位置原始值 = 输出轴角度 × 10 × 100
协议速度原始值 = 输出轴最大速度 dps × 10 × 100
```

## 硬件

接线和供电同 `06_电机基础控制`。

## 协议

```text
帧头：0x3E
命令：0xA4
数据长度：12
数据 0~7：int64_t angleControl，小端序，单位 0.01°
数据 8~11：uint32_t maxSpeed，小端序，单位 0.01dps
波特率：115200
```

## 串口命令

| 命令 | 功能 |
|------|------|
| `90` | 输出轴转到 90° |
| `p -45` | 输出轴转到 -45° |
| `v 90` | 设置输出轴最大速度为 90dps |
| `0` | 回到输出轴 0° |
| `+90` / `-90` | 相对当前目标增减 90° |
| `t` | 停止 |
| `r` | 读取多圈角度 |
| `h` | 帮助 |

## 使用步骤

1. 接线、供电同 06。
2. 上传程序，打开串口监视器。
3. 输入 `v 60` 设置较低移动速度。
4. 输入 `30`、`60`、`90` 逐步测试位置控制。
5. 输入 `t` 停止。
