# 机器人整合控制：两个 RS485 电机 + 四个 PWM 舵机

本示例用于最终机器人工程的主程序模板，演示同一块 ESP32-S3 同时控制：

- 2 个 MG4010-i10 RS485 电机，默认 ID 为 1 和 2
- 4 个 PWM 舵机，默认使用 GPIO10、GPIO11、GPIO12、GPIO13

## 接线

RS485 模块：

| ESP32-S3 | RS485 模块 |
|----------|------------|
| GPIO17 | DI |
| GPIO18 | RO |
| GPIO16 | DE 和 RE |
| GND | GND |

电机：

| RS485 模块 | MG4010-i10 |
|------------|------------|
| A | A/H |
| B | B/L |

舵机信号线：

| 舵机 | ESP32-S3 |
|------|----------|
| 舵机 1 | GPIO10 |
| 舵机 2 | GPIO11 |
| 舵机 3 | GPIO12 |
| 舵机 4 | GPIO13 |

供电要求：

- 电机使用独立 12V~24V 电源。
- 舵机使用独立 5V 电源。
- ESP32 GND、RS485 GND、电机电源负极、舵机电源负极必须共地。

## 串口命令

打开串口监视器，波特率设为 `115200`，行尾选择“换行”或“Both NL & CR”。

| 命令 | 作用 |
|------|------|
| `run` | 两个电机进入运行状态 |
| `stop` | 两个电机停止 |
| `v 30 30` | 左右电机输出轴速度设为 30 dps |
| `v 30 -30` | 左右电机反向，机器人原地转向 |
| `s 1 90` | 舵机 1 转到 90° |
| `pose 90 90 90 90` | 四个舵机分别转到指定角度 |
| `home` | 四个舵机回到 90°，两个电机速度设为 0 |

## 移植说明

前面 03、05、06、07、08、09 例程用于学习单个模块。最终工程建议以本示例为基础：

- 保留非阻塞舵机更新结构。
- 电机命令函数保留 `id` 参数。
- 左右电机速度命令按顺序发送到 ID 1 和 ID 2。
- 网页控制或传感器逻辑应加入主循环调度，不要使用长时间 `delay()`。
