## 📄 数据手册 (datasheets/)

### ESP32-S3 开发板

- **ESP32-S3原理图.pdf**

  - 技术参数（电压、电流、转速、扭矩）

  - 使用注意事项

- **MG4010-i10-R485协议.pdf**  
  电机的通信协议文档：
  - RS485 接线方法

## 🔌 驱动程序 (drivers/)

### CH341SER.EXE

WCH CH340/CH341 USB 转串口芯片的 Windows 驱动。

**什么时候需要装**：
- 插上板子，设备管理器里没有 COM 口
- 设备管理器里出现黄色感叹号的未知设备
- 提示"无法识别的 USB 设备"

**怎么装**：
1. 双击 `CH341SER.EXE`
2. 点"安装"
3. 拔掉板子重新插上
4. 设备管理器里应该能看到 `USB-SERIAL CH340 (COMx)`

**注意**：
- Windows 10/11 一般能自动识别，不需要手动装
- 如果板子用的是 CH343 芯片，需要装 CH343 的驱动（不在这个包里，去 [WCH 官网](https://www.wch.cn/downloads/CH343SER_EXE.html) 下）
- macOS 用户看 `docs/02_install_macos.md`

## 🎨 3D 模型 (models/)

### MG4010-i10.STEP

MG4010 电机的三维模型，STEP 格式。

**用来干嘛**：
- 设计机器人或机械结构时，确认电机安装位置和尺寸
- 检查螺丝孔位置对不对
- 做机械仿真
- 3D 打印配套的固定支架

**怎么打开**：
- FreeCAD（免费开源）
- Fusion 360（学生免费）
- SolidWorks
- Onshape（在线版，免费）

**不需要的情况**：
- 只是写代码控制电机，不用管 3D 模型
- 学校实验课一般用不到
